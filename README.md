# How to become data analytics.

- Used in technical interviews.
### <u>2. Business intelligent tools.</u>
- Example PowerBI/Tableau 
- Combination of SQL and PowerBI.
### <u>3. Excel</u>
- Use different for data analytics
- Used to store clean data.
- Fundamental skill
### <u>4. Python</u>
- Mostly used
